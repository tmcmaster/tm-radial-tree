# \<tm-radial-tree>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i tm-radial-tree
```

## Usage
```html
<script type="module">
  import 'tm-radial-tree/tm-radial-tree.js';
</script>

<tm-radial-tree></tm-radial-tree>
```
